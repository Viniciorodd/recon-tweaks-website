FAQ

Will this get me banned in Fortnite or VALORANT?
No. Every tweak operates at the Windows OS level — registry values, services, and power settings. Nothing touches game files, memory, or anti-cheat processes. Riot Vanguard, Easy Anti-Cheat, and BattlEye all operate independently of these optimizations. These same settings are used by professional players.
Is it safe to disable Spectre and Meltdown mitigations?
On a personal gaming PC you control, yes. The patches were designed to protect shared cloud servers from cross-process attacks. On a single-user gaming machine where you run the only code, the real-world risk is negligible and the performance gain is measurable.
My antivirus flagged the file. Is it safe?
Yes. Antivirus tools flag optimization apps because they modify registry keys and disable services — the exact actions the tool is designed to perform. This is a false positive. The app is built with Electron, a trusted framework. You can submit the file to Microsoft at microsoft.com/wdsi/filesubmission to whitelist it permanently.
Does it work on Windows 10?
Yes. Compatible with Windows 10 version 1903 and later, and all Windows 11 versions. Win11-specific features (Widgets, Teams Chat, News Feed) silently skip on Windows 10 without any error.
How does the server routing work exactly?
The background optimizer polls your running processes every 10 seconds. When it detects a supported game it pings all regional servers for that game with 3 packets each, sorts by average latency, flushes your DNS cache, and sets Cloudflare 1.1.1.1 as primary DNS to ensure the fastest possible resolution path. A Windows notification confirms the best server found.
Can I run it multiple times?
Yes. Licensed machines skip the key screen automatically. Running it again reapplies all tweaks — useful after a Windows Update that resets some settings. A new restore point is created each run.
